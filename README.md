## Getting Started

### Info

My node version - v16.14.3 
* npm
  ```sh
  npm node --version
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ThePiteroXx/movies-web.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your API KEYS config in `.env`
   ```env
    REACT_APP_API_TOKEN=<your_token(themoviedb.org)>
   ```
4. Open local server
    ```sh
   npm run start
   ```
